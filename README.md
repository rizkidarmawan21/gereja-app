# Aplikasi Pendaftaran Ibadah Gereja

## Tech

- Laravel 8
- Bootstrap 4.*

## How to use ?
- Clone this repo
- Run composer install
- Create Database
- Copy `.env.example` paste `.env`
- Setup database in `.env`
- Migrate database run php artisan migrate
- Running `php artisan serve`
